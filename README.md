# Chat-with-emojis
A realtime chat with nodejs.

### Installation
``` sh
    git clone https://github.com/moisesdelacruz/Chat-with-emojis.git
    cd Chat-with-emojis
    npm install
    npm start
```
Go To [http://localhost:3000/][dill]
